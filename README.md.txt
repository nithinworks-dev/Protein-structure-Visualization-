# 🧬 Protein Structure Visualization Project

## 📌 Objective
Visualize and analyze 3D protein structures using PyMOL.

## 🧰 Tools Used
- PyMOL
- RCSB Protein Data Bank

## 📂 Files Included
- Protein images (PNG)
- Project report
- PDB file
- PyMOL session file

## 🧪 Procedure
1. Downloaded protein structure from PDB
2. Opened in PyMOL
3. Applied visualization styles
4. Highlighted specific region

## 📊 Result
Successfully visualized protein structure and analyzed its components.

## 📚 Conclusion
Protein visualization helps understand structure and function of biomolecules.
